# 闪耀教育平台 - 微信小程序前端

一个现代化、动效丰富的教育评价平台微信小程序。

## 项目简介

闪耀教育平台是一个面向家长、学生和教师的教育评价系统。通过现代化的UI设计和流畅的动画效果，为用户提供优质的使用体验。

## 主要功能

### 三种角色

1. **家长端**
   - 查看孩子的课后评价
   - 浏览历史评价和照片
   - 查看成长趋势图表
   - 实时关注孩子表现

2. **学生端**
   - 查看自己的课后评价
   - 查看成长记录和积分
   - 浏览老师上传的照片
   - 查看学习统计数据

3. **教师端**
   - 对学生进行课后评价
   - 给学生打分
   - 上传课堂照片
   - 管理课程和班级

## 技术特点

### 现代化UI设计
- 渐变色背景（紫色、粉色、蓝色主题）
- 卡片式布局
- 毛玻璃效果（backdrop-filter）
- 圆角设计和阴影效果

### 丰富的动画效果
- **淡入动画（fadeIn）**: 页面元素加载时的淡入效果
- **上滑动画（slideUp）**: 卡片从底部滑入
- **缩放动画（scale）**: 统计数据的缩放展示
- **弹跳动画（bounce）**: 按钮和徽章的弹跳效果
- **交互动画**: 点击、滑动时的平滑过渡

### 响应式设计
- 自适应不同屏幕尺寸
- 灵活的网格布局
- 流畅的滚动体验

## 设计理念

### 色彩系统
- **紫色主题**: 学生端、角色选择
- **粉色主题**: 教师端
- **蓝色主题**: 家长端
- **辅助色**: 绿色、橙色用于状态标识

### 动画原则
- **有意义**: 每个动画都服务于用户体验
- **流畅**: 使用ease-out和ease-in-out缓动
- **适度**: 动画时长控制在0.3-0.8秒
- **性能优先**: 使用transform和opacity优化性能

### 交互设计
- **视觉反馈**: 所有可点击元素都有active状态
- **加载动画**: 使用错开的动画提升感知速度
- **空状态**: 优雅的空数据展示
- **下拉刷新**: 支持页面数据刷新
